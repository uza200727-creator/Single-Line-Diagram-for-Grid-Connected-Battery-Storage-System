# Grid-Connected Battery Storage System (Single Line Diagram)

This project demonstrates a simplified single-line diagram of a grid-connected battery energy storage system created using AutoCAD. The system takes in AC power, converts it to a lower usable voltage, sends it to a central busbar that distributes the power to 2 AC/DC inverters that change the AC power to DC power, and uses it to charge the 2 batteries connected to the inverters.

[Single Line Diagram](singlelinediagram.png)

## Description of Diagram

The diagram contains the following commponents from top to bottom:

-AC Power Supply (Connection to Grid)
-Circuit Breaker
-Step-Down Transformer (reduces AC voltage to a lower safer amount for batteries)
-Busbar (distributes power among the 2 batteries)
-2x AC/DC Power Inverter
-2x Batteries 

## Skills Demonstrated

AutoCAD: Block creation, layer management, AutoCAD command usage, usage of standardized power symbols, simple shape sketching
